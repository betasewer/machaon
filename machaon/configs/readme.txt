machaon フォルダ
設定情報を配置するフォルダです。

/ apps.ini
machaonから呼び出す外部アプリの情報を書いておきます。apps.iniを参照してください。
/ guid.ini [win]
machaonで利用するデフォルト以外のguidを書いておきます。

/ packages
ダウンロードしたパッケージが置かれます。
/ packages / packages.ini
パッケージのバージョン情報などが記録されます。

/ credential
リモートリポジトリの認証情報を配置します。
/ credential / ***.ini
認証情報を書いておきます。credential.iniを参照してください。

/ store
machaonスクリプトを配置します。
Stored型で読み込んだり保存したりできます。

/ local
アプリごとの設定情報を配置するフォルダです。
サブディレクトリがアプリ一つに対応します。
